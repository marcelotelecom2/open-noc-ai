# Code of Conduct

## Our Commitment

This project is committed to providing a respectful, inclusive and collaborative environment for everyone.

We aim to create a community where all participants feel welcome and valued.

---

## Expected Behavior

* Be respectful and constructive
* Focus on technical collaboration
* Welcome different perspectives
* Provide feedback professionally
* Act with integrity and responsibility

---

## Unacceptable Behavior

* Harassment or discrimination of any kind
* Offensive, abusive or hostile language
* Personal attacks or insults
* Trolling or disruptive behavior
* Publishing sensitive or private information

---

## Enforcement

Project maintainers are responsible for enforcing this code of conduct.

They may:

* Remove comments or contributions
* Close issues or pull requests
* Restrict participation if necessary

---

## Reporting Issues

If you experience or witness unacceptable behavior, please report it to the project maintainers.

All reports will be handled with confidentiality and seriousness.

---

## Scope

This Code of Conduct applies to all project spaces, including:

* GitHub issues and pull requests
* Discussions and documentation
* External communications representing the project

---

## Final Note

By participating in this project, you agree to follow this Code of Conduct and contribute to a positive and professional environment.
