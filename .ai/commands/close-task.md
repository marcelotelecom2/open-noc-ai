# Command — Close Task

## Task

Generate a task closing report.

## Required Sections

Include:

## Objective

What the task intended to accomplish.

## Scope

What was included.

## Work Completed

What was implemented or validated.

## Files Changed

List affected files.

## Validation

List tests/checks performed.

## Issues Found

List errors, risks or inconsistencies.

## Architecture Notes

Mention relevant architectural decisions.

## Next Steps

List remaining actions.

## Final Status

Use one:

- Completed
- Completed with pending validation
- Blocked
- Rejected
